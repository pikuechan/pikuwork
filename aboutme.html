ABOUT ME

1988年 茨城県日立市生まれ。慶應義塾大学 環境情報学部（SFC）在学中に、アプリケーション開発やプロダクトデザインについて学ぶ。モノ作りのモットーはヒトが心地よくつかえるモノを作ることで、世の中のあらゆるものを、わかりやすく、つかいやすく、心地よくつかい続けられるものにしたいと思っている。特に情報をわかりやすくデザインすることが得意で、情報設計(IA)やユーザー体験(UX)について考えるのが好き。趣味は かわいいものをつくることで、たまにクラフトアートやフードアートで遊んだりしている。

INTEREST KEY WORDS User interface(UI) User experience(UX) Information architecture(IA) Infographics Semantic web
