HTML/CSS (Scss)

OOCSSやBEMなどのCSS設計はまだ不慣れな部分もあるが、基本知識はある（このページを作れる程度）。前職では、プロトタイプの作成などで業務利用していた。

JavaScript (jQuery)

あまり利用する機会がなく実践経験に乏しいが、デザイン調整に関連する基本動作であれば、調べながら使うことができる。

Git

SourceTreeなどのクライアントソフトを利用すれば、基本機能はすべて利用することができる。チームでのソース管理経験は浅いため、一部使い慣れていない機能もある。

Adobe

illustrator,Photoshop,AfterEffectsの基本機能は使うことができる。XD,InDesignは少し触ったことがある程度。それ以外はほぼ利用経験なし。

Sketch/Figma

サービス画面のUI作成はほとんどこの２ツールをメインに利用している。（グラフィック作成はAdobeがメイン）基本機能はすべて利用することができる。

Prototyping

PC/Mobile/App(Android/iOS)各環境に合わせたプロトタイピングをすることができる。プロトタイピングツールでの作成がメインだが、Webの場合はコードベースでのプロトタイピングをすることもある。

Graphic/Illustration

シンプルかつ簡単なグラフィック・イラスト作成は行うことができるが、複雑なグラフィック・イラスト作成は経験がなく、あまり得意ではない。

Animation/Motion Design

UI表現に用いるようなシンプルなアニメーション＆モーションデザインであれば作成することができる。プロトタイピングツールや、AfterEffectsなどを用いて作成する。

Interview & Testing

市場調査や価値検証を目的としたインタビュー調査から、ユーザビリティ評価検証まで、目的に応じたインタビュー＆テスト設計と、サービスへのフィードバックを行うことができる。

Data analysis

GoogleAnalyticsを利用したユーザー調査＆行動分析を行うことができる（基本閲覧機能のみ）。SQL等を用いた分析業務の経験はないが、過去に少し学んだ経験はある。

Servive Design

基本設計は Human Centered Design(HCD)のフレームワークを参考にすることが多いが、Design Sprintや、Hooked Modelなど、トレンドに沿ってあらゆる思考プロセスを参考にするようにしている。
